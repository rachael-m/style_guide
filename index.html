Rachael's Website Style Guide

Color Scheme

This color scheme has been checked to be colorblind safe.

Deep Blue Sea
RGB #081345
Deep Blue Sea will be used for headers, logos, and as an accent.

Cool Blue
RGB #B1BBDF
Cool Blue will be used as a background color.

Royal Blue
RGB #1837C4
Royal Blue will be used for links and buttons.

Slate-Gray Blue
RGB #43444A
Slate-Gray Blue will be used for inactive links/buttons.
